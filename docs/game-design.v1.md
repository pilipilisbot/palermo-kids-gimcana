# Game Design v1 — Prova cooperativa única

Cada punt dona una peça (número/paraula/símbol). Al final, l'equip combina totes les peces per obtenir el **Codi del Tresor**.

## Regles
- Tothom ha de participar (petits observen, grans dedueixen).
- Si una peça falla, l'equip pot demanar 1 pista extra.
- Premi final: gelat o berenar.

## Pendent
- Tancar proves exactes per punt
- Definir format de pistes (text curt o QR)
- Validació amb recorregut real (temps i cansament)
