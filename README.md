# Palermo Kids Gimcana

Gimcana familiar (6–12 anys) per Palerm amb ruta curta, proves cooperatives i final amb premi.

## Objectiu
- Ruta 1.8–2.2 km, 60–120 min.
- Una única prova cooperativa final (codi de tresor).
- Adaptada a 2 famílies amb nens de 6 a 12.

## Punts proposats (v1)
1. Teatro Massimo
2. Quattro Canti
3. Fontana Pretoria
4. Piazza Bellini (Martorana / San Cataldo)
5. Cattedrale di Palermo
6. Piazza Marina (ficus gegants, final + premi)
